
# 📘 DevOps Project — CI/CD Pipeline with Jenkins, Terraform, Docker & AWS

פרויקט זה מציג תהליך DevOps מלא הכולל פיתוח אפליקציית Python FastAPI, בניית Docker image, בדיקות אוטומטיות, פריסה ל‑AWS EC2, תשתיות ב‑Terraform, ו‑CI/CD מלא באמצעות Jenkins.

## 🚀 1. Project Overview
המערכת כוללת:
- אפליקציית FastAPI
- Dockerfile רב־שלבי
- בדיקות pytest
- Jenkins Pipeline מלא
- Terraform שמקים EC2, IAM, S3, Security Groups
